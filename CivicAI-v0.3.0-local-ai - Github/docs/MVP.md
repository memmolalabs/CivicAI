# MVP 0.1

## Flusso dimostrabile

1. Selezione volontaria di un testo.
2. Menu contestuale “Analizza con CivicAI”.
3. Apertura del pannello laterale.
4. Estrazione strutturata.
5. Correzione manuale.
6. Copia della bozza.

## Fuori ambito

- scraping;
- analisi automatica dei feed;
- invio automatico a Comuni o PEC;
- acquisizione automatica di fotografie;
- verifica fattuale delle accuse;
- geolocalizzazione invisibile;
- profilazione degli utenti.
